is_china_ip
===========

get an IP location whether it is in China

###How to use?
```
var is_china_ip = require('is_china_ip');
var result = is_china_ip(IP_STRING);
```
###Another method
* ip2long 

   `var iplong = is_china_ip.ip2long(IP_STRING);`
* long2ip

	`var ip = is_china_ip.long2ip(IP_NUMBER);`
